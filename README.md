# Password Strength Checker

## Description
This project is a Python-based Password Strength Checker. It analyzes a password based on:
- Minimum 8 characters
- Uppercase letter
- Lowercase letter
- Number
- Special character

The program classifies passwords as Weak, Medium, or Strong and provides suggestions to improve password security.

## Technologies Used
- Python 3

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Displays password strength
- Suggests improvements for weak passwords
